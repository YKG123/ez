1. Two Pointers

Use When
  • Sorted arrays
  • Palindromes
  • Pair/triplet sums
Core Idea

Move pointers intelligently instead of nested loops.

Template
let l = 0; let r = arr.length - 1; while (l < r) { let sum = arr[l] + arr[r]; if (sum === target) { return [l, r]; } if (sum < target) l++; else r--; }
Example Problems
  • Two Sum II
  • 3sum (sort the array then do 2sum II)
  • Container With Most Water
  • Best time to buy and sell stock
  • Trapping Rain Water

21. Bucket Sort

Use When
  • cursed problems with counting frequency or sorting positive integers in sub nlogn time
Core Idea

Create n buckets and count how many times each int appears in that bucket.

Some problems will want you to skip the rebuild step.

Template
/* bucket sort for non-negative integers */ if (nums.length === 0) return []; /* find max value */ const max = Math.max(...nums); /* create buckets */ const buckets = Array(max + 1).fill(0); /* count frequencies */ for (const num of nums) { buckets[num]++; } /* rebuild sorted array */ const result = []; for (let i = 0; i < buckets.length; i++) { while (buckets[i]> 0) { result.push(i); buckets[i]--; } } return result;
Example Problems
  • Top K Frequent Elements
  • Contains Duplicate III
  • Maximum Gap
  • Sort Characters By Frequency
  • Top K Frequent Words
