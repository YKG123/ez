LeetCode Common Patterns Cheat Sheet

1. Two Pointers

Use When
  • Sorted arrays
  • Palindromes
  • Pair/triplet sums
Core Idea

Move pointers intelligently instead of nested loops.

Template
let l = 0; let r = arr.length - 1; while (l < r) { let sum = arr[l] + arr[r]; if (sum === target) { return [l, r]; } if (sum < target) l++; else r--; }
Example Problems
  • Two Sum II
  • Container With Most Water
  • Trapping Rain Water

2. Sliding Window

Use When
  • Subarrays/substrings
  • Contiguous ranges
  • Longest/shortest sequences
Core Idea

Expand and shrink a window dynamically.

Template
let l = 0; for (let r = 0; r < arr.length; r++) { while (windowInvalid) { l++; } result = Math.max(result, r - l + 1); }
Example Problems
  • Longest Substring Without Repeating
  • Minimum Window Substring

3. Hash Map

Use When
  • Fast lookups
  • Counting frequencies
  • Duplicates/anagrams
Core Idea

Store previous information for O(1) access.

Template
const map = new Map(); for (let i = 0; i < nums.length; i++) { let needed = target - nums[i]; if (map.has(needed)) { return [map.get(needed), i]; } map.set(nums[i], i); }
Example Problems
  • Two Sum
  • Group Anagrams

4. Binary Search

Use When
  • Sorted arrays
  • Monotonic conditions
Core Idea

Cut search space in half each step.

Template
let l = 0; let r = arr.length - 1; while (l <= r) { let mid = Math.floor((l + r) / 2); if (arr[mid] === target) return mid; if (arr[mid] < target) l = mid + 1; else r = mid - 1; }
Example Problems
  • Search Rotated Array
  • Koko Eating Bananas

5. DFS

Use When
  • Trees
  • Graphs
  • Backtracking
Core Idea

Go deep before backtracking.

Template
function dfs(node) { if (!node) return; dfs(node.left); dfs(node.right); }
Example Problems
  • Max Depth Binary Tree
  • Number of Islands

6. BFS

Use When
  • Shortest path
  • Level-order traversal
Core Idea

Process nodes level by level.

Template
let queue = [start]; while (queue.length) { let node = queue.shift(); for (let neighbor of node.neighbors) { queue.push(neighbor); } }
Example Problems
  • Word Ladder
  • Rotting Oranges

7. Dynamic Programming

Use When
  • Repeated subproblems
  • Optimization/counting
Core Idea

Reuse previous computations.

Template
const dp = Array(n).fill(0); dp[0] = baseCase; for (let i = 1; i < n; i++) { dp[i] = dp[i - 1] + something; }
Example Problems
  • Climbing Stairs
  • Coin Change

8. Heap / Priority Queue

Use When
  • Top K
  • Repeated min/max
Core Idea

Efficiently maintain order.

Example Problems
  • Kth Largest Element
  • Merge K Sorted Lists

9. Monotonic Stack

Use When
  • Next greater/smaller element
  • Histogram problems
Core Idea

Maintain increasing/decreasing order in stack.

Template
const stack = []; for (let i = 0; i < arr.length; i++) { while ( stack.length && arr[i] > arr[stack[stack.length - 1]] ) { stack.pop(); } stack.push(i); }
Example Problems
  • Daily Temperatures
  • Largest Rectangle Histogram
